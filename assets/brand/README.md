# Brand Assets

This folder is the source of truth for bun-forge brand images.

## Files

- `bun-forge-logo-master.png`: original high-resolution generated logo. Keep unchanged.
- `bun-forge-logo-full-1024.png`: full logo with wordmark for large surfaces.
- `bun-forge-logo-full-640.png`: full logo with wordmark for README/npm usage.
- `bun-forge-mark-1024.png`: icon mark without wordmark.
- `bun-forge-mark-512.png`: repository avatar, package icon, docs cards.
- `bun-forge-mark-192.png`: PWA-style icon size.
- `bun-forge-apple-touch-180.png`: Apple touch icon size.
- `bun-forge-mark-mono-1024.png`: monochrome/silhouette check.
- `bun-forge-favicon.svg`: simplified favicon source. Use for tiny sizes.
- `bun-forge-favicon-base-512.png`: raster source for favicon exports.
- `bun-forge-favicon-64.png`: 64 px favicon preview/export.
- `bun-forge-favicon-48.png`: 48 px favicon preview/export.
- `bun-forge-favicon-32.png`: 32 px favicon preview/export.
- `bun-forge-favicon-16.png`: 16 px favicon preview/export.
- `favicon.ico`: legacy favicon bundle with 16, 32, and 48 px entries.
- `bun-forge-social-preview-1280x640.png`: GitHub/Open Graph preview draft.
- `preview.html`: local browser preview for reviewing logo usage.
- `preview-screenshot.png`: latest screenshot of the preview page.

## Rules

- Do not downscale the full illustrated logo for favicon usage.
- Use the full logo only when the wordmark stays readable.
- Use the mark without wordmark for avatars and square icons.
- Use the dedicated favicon source for 16-64 px surfaces.
- Keep the master image untouched; create new versioned files for iterations.
